InsightFlow AI is an AI-powered data analytics automation built using n8n that converts raw CSV files into meaningful visual insights without manual chart configuration.

The system ingests structured CSV data, automatically analyzes column types and distributions, applies appropriate aggregations, and uses an LLM-based reasoning layer to determine the most suitable chart type (bar, pie, line, or comparative). Charts are generated dynamically via an HTTP visualization API, enabling instant insight generation from arbitrary datasets.

Unlike traditional CSV visualization tools that rely on hard-coded logic or predefined schemas, InsightFlow AI is designed as a reusable, data-agnostic analytics engine. It adapts to new datasets without workflow changes, making it suitable for real-world reporting, KPI monitoring, and automated analytics pipelines.

The project demonstrates practical skills in workflow orchestration, AI agent integration, data transformation, and API-driven visualization, reflecting production-style automation rather than a static demo.
